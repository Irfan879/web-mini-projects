# Delta Demo
Gameplay and variants.
A command starting with "Simon says" means that the players must obey that command.
A classic memory game where you repeat an ever-growing sequence of colored lights and sounds. Each round adds one more step — stay sharp and see how far you can go!


# How to Play
Watch the sequence of flashing colors carefully
Repeat the sequence by clicking the buttons in the correct order
Each round adds one more color to the sequence
One wrong move and the game resets — try to beat your high score!
