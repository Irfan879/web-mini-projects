# 🎵 Spotify Clone

A responsive one-page clone of the Spotify Web Player UI,
built with pure HTML and CSS — no JavaScript, no frameworks.

## ✨ Features
- Spotify-inspired dark theme UI
- Sidebar with navigation links and playlists
- Main content area with music cards
- Fixed bottom music player bar
- Hover effects and smooth transitions
- Fully responsive layout

## 🛠️ Built With
- HTML5
- CSS3
- Flexbox & Grid
